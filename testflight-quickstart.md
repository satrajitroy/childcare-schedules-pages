# ChildCare Schedules — TestFlight Quick Start

This beta is distributed through Apple TestFlight.

## Install

1. Install Apple’s **TestFlight** app from the App Store.
2. Open the invite link you received.
3. Tap **Accept**.
4. Install **ChildCare Schedules**.

## Try the sample schedule

1. Open ChildCare Schedules.
2. Go to **Staff**.
3. Set **Number of teachers/core staff to use** and **Number of floaters to use**.
4. Go to **Rooms** and select the classrooms to use.
5. Go to **Breaks, Lunch & Prep** and choose the coverage mode.
6. Go to **Create** and create a schedule.
7. View and export from **Results**.

## Suggested demo runs

Start with a smaller run first, then increase size:

- 50 teachers/core staff + 0 floaters
- 70 teachers/core staff + 0 floaters
- 70 teachers/core staff + 10–20 floaters

As floaters, covered breaks, lunch, prep, and other choices are added, the exact-cover search space grows quickly. Larger runs may take several minutes.

## Important notes

- This beta runs locally on-device.
- No login is required.
- No scheduling data is sent to a server by the app.
- TestFlight builds expire. If your beta stops opening, install the latest TestFlight build or contact the developer.

## Feedback to send

When reporting an issue, please include:

- device model,
- iOS version,
- approximate staff count,
- approximate room count,
- number of floaters,
- whether break/lunch/prep coverage was enabled,
- whether the app returned no schedule, ran slowly, or closed unexpectedly.
