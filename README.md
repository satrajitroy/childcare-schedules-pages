# ChildCare Schedules — Help Docs Bundle

This bundle contains user-facing and partner-facing help documents for TestFlight, App Store review, GitHub Pages, or direct email.

## Files

- `index.html` — landing page linking to all docs.
- `user-manual.html` / `user-manual.md` — full user manual.
- `testflight-quickstart.html` / `testflight-quickstart.md` — short instructions for beta testers.
- `support.html` — support page with Rocq/soundness wording and links to docs.
- `privacy.html` — privacy page.
- `technical-integration-brief.html` / `.md` — partner/integration brief with Rocq soundness wording.
- `tester-email-template.txt` — text you can paste into an email with the TestFlight link.
